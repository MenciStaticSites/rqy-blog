BZOJ5329 [SDOI2018] 战略游戏

发表于 2018-05-17 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

一个 nnn 个点 mmm 条边的无向连通图，qqq 次询问。每次询问给出一个集合 SSS ，问有多少结点 xxx 满足：

  1. x∉Sx\not\in Sx​∈S;
  2. 删除结点 xxx 及与其相连的所有边之后， SSS 中的点不连通。

n,q⩽105;m,∑∣S∣⩽2×105n,q\leqslant10^5;m,\sum|S|\leqslant2\times10^5n,q⩽105;m,∑∣S∣⩽2×105 。

多组数据，数组组数 T⩽10T\leqslant10T⩽10 ；时限 10s10s10s 。

Solution

首先，删除某结点后 xxx 和 yyy 不连通，这等价于这个结点在所有 xxx 到 yyy 的简单路径上。

圆方树基本性质：原图上 xxx 到 yyy 的所有简单路径交即为圆方树上 xxx 到 yyy 的所有圆点。

那么我们建出圆方树；可以发现答案就是点集 SSS 建出的虚树中除 SSS 里所有点之外的圆点个数。

所以暴力建一遍虚树就可以了。时间复杂度 O(nlog⁡n+∑(∣S∣log⁡∣S∣))O(n\log n+\sum(|S|\log|S|))O(nlogn+∑(∣S∣log∣S∣)) （ RMQO(1)O(1)O(1)LCA ）。

考场上 yy 出一个正确的点双结果由于 RMQ 数组开小爆零了… 。
