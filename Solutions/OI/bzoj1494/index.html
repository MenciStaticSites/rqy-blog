BZOJ1494 [NOI2007] 生成树计数

发表于 2018-03-21 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

有一个 nnn 个点的图。nnn 个点排成一排，两个点之间有边当且仅当它们距离不超过 kkk （即它们之间隔了不超过 k−1k-1k−1 个点）。

求这个图的生成树个数。

n⩽1015,k⩽5n\leqslant10^{15}, k\leqslant5n⩽1015,k⩽5 。

Solution

题面上的矩阵树定理看上去很吸引人…然而你总不会想拿那个算吧…

可以发现如果有一棵生成树，只取出前 ttt 个点之后，它会变成至多 kkk 棵树所组成的森林，且每棵树都至少有一个点在最后 kkk 个里面（因为如果某棵树的所有点都不在最后 kkk 个里面的话，后面的点再也没法和它们连接）。

那么我们可以考虑以后 kkk 个点的连通性作为状态进行 DP 。

形式化的，我们把所有 wkw_kwk​ （贝尔数）种 kkk 个点的连通性标号以 0…wk−10\dots w_k-10…wk​−1 ，令 fi,j(i≥k)f_{i,j} (i \geq k)fi,j​(i≥k) 表示前 iii 个点连成森林，每棵树都有结点在后 kkk 个里面，且后 kkk 个点的连通性为 jjj （即标号为jjj 的集合划分方案）的方案数。

那么由于第 iii 个点只能向 i−k…i−1i-k\dots i-1i−k…i−1 连边， fif_ifi​ 只会依赖于 fi−1f_{i-1}fi−1​ 。

考虑转移。如果第 lll 个状态右面追加一个点并向左连边形成第 jjj 种状态的方法有 Al,jA_{l, j}Al,j​ 种，那么转移式即为 fi,j=∑lAl,jfi−1,lf_{i, j}=\sum_l A_{l,j}f_{i-1, l}fi,j​=∑l​Al,j​fi−1,l​ 。 AAA 可以通过枚举 lll 及 2k2^k2k 种连边方式求出（注意判断不合法的情况）。

考虑边界条件。显然 k>nk>nk>n 的时候没有作用，我们可以假定 k⩽nk\leqslant nk⩽n 。那么所有 fi,j(i<k)f_{i, j} (i<k)fi,j​(i<k) 都是没有作用（甚至没有定义）的；我们只需要确定 fk,jf_{k,j}fk,j​ 。由于这是前 kkk 个点之间都有边，所以把 jjj 这个状态中每个连通块当成完全图求出生成树个数再乘起来即可。

之后显然就可以矩阵快速幂加速了。

时间复杂度（零碎的直接抹去）为 O(wn3log⁡n)O(w_n^3\log n)O(wn3​logn) 。
