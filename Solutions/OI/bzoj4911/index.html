BZOJ4911 [SDOI2017] 切树游戏

发表于 2018-04-27 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

一棵 nnn 个结点的树，每个点有点权 aia_iai​ 。有 qqq 个操作：

  • Change x y, 表示把编号为 xxx 的点的点权改为 yyy;
  • Query k, 表示询问点权异或和为 kkk 的连通块个数  mod 10007\bmod 10007mod10007 。

n⩽30000,ai∈[0,m),m⩽128,q⩽30000n\leqslant30000,a_i\in[0, m),m\leqslant128,q\leqslant30000n⩽30000,ai​∈[0,m),m⩽128,q⩽30000 。

mmm 是 222 的幂。

Solution

先考虑没有修改时怎么做。

由于 m⩽128m\leqslant128m⩽128 ，我们可以求出每一种答案的值。 DP 即可。

令 fi,jf_{i,j}fi,j​ 表示以 iii 为根的子树中包含 iii 的异或和为 jjj 的连通块个数，则显然有

Fi=xai∏k∈soni(Fk+1)F_i=x^{a_i}\prod_{k\in son_i}(F_k+1) Fi​=xai​k∈soni​∏​(Fk​+1)

其中乘法为异或卷积，乘 xaix^{a_i}xai​ 表示下标对 aia_iai​ 取模（取 vfk 集合幂级数的含义）。这样就是 O(nm2)O(nm^2)O(nm2) 了。

考虑快速沃尔什-哈达玛变换，即

F(f)i=∑j=0m−1(−1)count(i&j)fj\mathcal F(f)_i=\sum_{j=0}^{m-1}(-1)^{count(i\&j)}f_j F(f)i​=j=0∑m−1​(−1)count(i&j)fj​

其中 &\&& 为按位与， countcountcount 统计二进制数中 111 的个数。容易证明若 a=b∗ca=b*ca=b∗c （异或卷积）则

F(a)=F(b)⋅F(c)\mathcal F(a)=\mathcal F(b)\cdot\mathcal F(c) F(a)=F(b)⋅F(c)

其中 ⋅\cdot⋅ 表示点乘，即逐位相乘。且我们有 F(F(f))i=m×fi\mathcal F(\mathcal F(f))_i=m\times f_iF(F(f))i​=m×fi​ ，于是逆变换很容易进行。

那么我们可以 DP 初值时就 FWT 一下，计算时全部直接点乘（ +1+1+1 现在要在每一位上都 +1+1+1 ）。最后求出所有 F(fi)\mathcal F(f_i)F(fi​) 的和再变换回来。复杂度是 O(nm+mlog⁡m)O(nm+m\log m)O(nm+mlogm) 。

再考虑有修改时怎么做。

考虑树剖维护 DP 。即树剖之后，每个点维护一个 HiH_iHi​ 表示只考虑轻儿子和自己的时候的 DP 值（也就相当于把重儿子切下来）。令 wiw_iwi​ 表示 iii 的重儿子，则

Fi=Hi(1+Fwi)=Hi+HiHwi(1+Fwwi)=Hi+HiHwi+⋯+Hi…Hw⋱iF_i=H_i(1+F_{w_i})=H_i+H_iH_{w_i}(1+F_{w_{w_i}})=H_i+H_iH_{w_i}+\dots+H_i\dots H_{w_{\ddots_i}} Fi​=Hi​(1+Fwi​​)=Hi​+Hi​Hwi​​(1+Fwwi​​​)=Hi​+Hi​Hwi​​+⋯+Hi​…Hw⋱i​​​

由于一条重链上的点在线段树中是连续的，所以 FiF_iFi​ 就等于这个区间的所有前缀的积的和。

同理，一条重链上的所有 FFF 的和就是这个区间内的所有子区间的积的和。

那么考虑修改时，修改一个点会影响到 O(log⁡n)O(\log n)O(logn) 条重链。直接在线段树上修改查询即可。

一个细节：去掉某个轻儿子对其父亲的影响时，可能会除零。所以说要维护每个点所有轻儿子中零的个数和非零部分的乘积。

时间复杂度 O(nm+qmlog⁡n)O(nm+qm\log n)O(nm+qmlogn) 。
