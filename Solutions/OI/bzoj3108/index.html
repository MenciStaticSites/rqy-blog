BZOJ3108 [CQOI2013] 图的逆变换

发表于 2018-03-14 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

定义一个图的变换：对于一个有向图G=(V,E)G=(V, E)G=(V,E)，建立一个新的有向图：

V′={ve∣e∈E}V'=\{v_e|e \in E\}V′={ve​∣e∈E}，E′={(vb,ve)∣b=(u,v),e=(v,w)}E'=\{(v_b, v_e)|b=(u,v), e=(v,w)\}E′={(vb​,ve​)∣b=(u,v),e=(v,w)}，G′=(V′,E′)G'=(V', E')G′=(V′,E′)。

也就是说每个边变成一个点，如果边b的终点和边e的起点相同则b到e连一条边。

现在给定G′G'G′，问是否存在GGG。G′G'G′的点数不超过300300300。

Solution

如果G′G'G′中有(u,w),(v,w)(u,w), (v,w)(u,w),(v,w)两条边，那么说明GGG中u,vu,vu,v的终点相同；那么G′G'G′中u,vu,vu,v连到的点应该是一样的。

也就是说，如果我在G′G'G′中令SiS_iSi​表示iii连到的点集，那么Si=SjS_i=S_jSi​=Sj​和Si∩SjS_i\cap S_jSi​∩Sj​必有一成立。

反之，如果上述条件成立，我可以把所有点按照SSS划分，即可得到GGG中每个点的出边集合；然后容易找出GGG中每个点的入边集合，易证这个GGG是合法的。

于是bitset求出SSS之后枚举i,ji,ji,j判断即可。
