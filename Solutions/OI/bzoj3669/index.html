BZOJ3669 [NOI2014] 魔法森林

发表于 2018-03-19 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

一个 nnn 个点 mmm 条边的无向图，每条边有 AAA 和 BBB 两个边权。求一个从 111 到 nnn 的路径 PPP ，最小化 max⁡e∈PAe+max⁡e∈PBe\max_{e \in P} A_e + \max_{e \in P} B_emaxe∈P​Ae​+maxe∈P​Be​。

n⩽50000,m⩽100000,Ai,Bi⩽50000n \leqslant 50000, m \leqslant 100000, A_i, B_i \leqslant 50000n⩽50000,m⩽100000,Ai​,Bi​⩽50000.

Solution

如果我们固定 maxAmax_AmaxA​ ，那么显然可以走所有 AAA 不超过 maxAmax_AmaxA​ 的边。把这些边拿出来按 BBB 求最小生成树，那么 maxBmax_BmaxB​ 就是从 111 到 nnn 的树上路径上的 BBB 的最大值。

于是可以按 AAA 对所有边排序，依次加入；利用 LCT 动态维护按 BBB 的最小生成树即可。
