BZOJ5248 [多省联考2018] 一双木棋

发表于 2018-04-08 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

有一个 n∗mn * mn∗m 的方格， Alice 和 Bob 玩游戏。每次每人可以选择一个格子占领，前提是这个格子未被占领且它左上方的所有格子都已被占领。

第 iii 行第 jjj 列的格子若被 Alice 占领则 Alice 获得 Ai,jA_{i, j}Ai,j​ 分，若被 Bob 占领则 Bob 获得 Bi,jB_{i, j}Bi,j​ 分。 Alice 先手，所有格子都被占领时结束。双方都想最大化自己的得分与对方得分的差。求双方采取最优策略时 Alice 得分与 Bob 得分的差。

n,m⩽10n, m\leqslant10n,m⩽10 。

Solution

首先显然可以爆搜。就是说每次枚举所有的走法，然后如果这步是 Alice 则取得分差的 max ，否则取 min 。

然后记忆化一下（扔个 map 什么的），这题就过了。

为什么呢？

考虑游戏状态。显然第 i 行选了第一个开始的连续若干个。 假设第 iii 行选了 aia_iai​ 个，那么肯定有 m≥a1≥a2≥⋯≥an≥0m \ge a_1 \ge a_2 \ge \dots \ge a_n \ge 0m≥a1​≥a2​≥⋯≥an​≥0 。而且剩余的游戏状态由所有 aaa 唯一决定。

我们令 $ x_0 = m - a_1, x_1 = a_1 - a_2, x_2 = a_2 - a_3, …, x_n = a_n $ ，那么所有 xxx 非负。状态数就是 ∑i=0nxi=m\sum_{i=0}^n x_i = m∑i=0n​xi​=m 的非负整数解数，利用隔板法可知方案数为 (n+mm)n + m \choose m(mn+m​) 。极限数据下不到 2e5 。
