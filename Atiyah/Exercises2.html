Atiyah 第二章习题解答

Solutions to the exercises of Introduction to Commutative Algebra, Atiyah, Chapter 2.

Modules

Exercise 1. Show that (Z/mZ)⊗Z​(Z/nZ)=0, if m,n coprime.

Proof. If m,n coprime then n is a unit in Zm​, so

x⊗y=n−1x⊗ny=0

Hence (Z/mZ)⊗Z​(Z/nZ)=0, ’cause it’s generated by all x⊗y.

□

Exercise 2. Let A be a ring, a an ideal, M an A-module, show that (A/a)⊗A​M is isomorphic to M/aM.

Proof. Obviously a→A→A/a→0 is an exact sequence, so is a⊗M→A⊗M→(A/a)⊗M→0. But a⊗M≅aM and A⊗M≅M, and the first arrow is the inclusion map, so (A/a)⊗M≅(M/aM).

□

Exercise 3. Ler A be a local ring, M and N finitely generated A-modules. Prove that if M⊗N=0, then M=0 or N=0.

Proof. Let m be the maximal ideal of A and k=A/m be the residue field of A. Let Mk​ denote k⊗A​M=(M/mM), then by Nakayama Lemma, Mk​=0→M=0. So we have M⊗A​N=0⟹(M⊗A​N)k​=0⟹Mk​⊗k​Nk​=0. But Mk​ and Nk​ are vector spaces over field k, so Mk​⊗k​Nk​=0 implies Mk​=0 or Nk​=0, hence M=0 or N=0.

□

Exercise 4. Let Mi​(i∈I) be any family of A-modules, and let M be their direct sum. Prove that M is flat ⟺ each Mi​ is flat.

Proof. M=⨁i∈I​Mi​ is flat ⟺ for all injective f:N→N′, f⊗(⨁i∈I​1Mi​​)=⨁i∈I​(f⊗1Mi​​):N⊗M→N′⊗M is injective. And ⨁i∈I​fi​ is injective if and only if each fi​ is injective, so qed.

□

Exercise 5. Ler A[x] be the ring of polynomials in one indeterminate over a ring A. Prove that A[x] is a flat A-algebra.

Proof. As a A-module, A[x]≅⨁n=0∞​A, so by Exercise2.4 A[x] is flat (since A is flat).

□

Exercise 6. For any A-module M, let M[x] denote the set of all polynomials in x with coefficients in M. Defining the product of an element of A[x] and an element of M[x] in the obvious way, show that M[x] is an A[x]-module.

Show that M[x]≅A[x]⊗A​M.

Proof. By define (∑i=0n​ai​xi)(∑j=0k​mj​xj)=∑i=0n​∑j=0k​ai​mj​xi+j, trivially the module axioms hold here.

Consider a map ϕ:M[x]→A[x]⊗A​M defined by ϕ(mxi)=xi⊗m, then it’s a well-defined A[x]-module homomorphism. If we define ψˉ​:A[x]×M→M[x] by ψˉ​(∑i​ai​xi,m)=∑i​(ai​m)xi, then it’s clearly A-bilinear, so induces an A-module homomorphism ψ:A[x]⊗A​M→M[x]. It’s easy to prove ϕ and ψ are inverses, so A[x]⊗A​M≅M[x].

□

Exercise 7. Let p be a prime ideal in A, show that p[x] is a prime ideal in A[x]. If m is a maximal ideal in A, is m[x] a maximal ideal in A[x]?

Proof. Consider map ϕ:A[x]→(A/p)[x], then Kerϕ=p[x]. Then p[x] is prime since (A/p)[x] is an integral domain.

If m is maximal, then (A/m)[x] doesn’t have to be a field, so m[x] is not maximal in general. For a counterexample, 2Z is a maximal ideal in Z, but (2Z)[x]⊆(2,x) is not maximal.

□

Exercise 8.

1.

If M and N are flat A-modules, then so is M⊗A​N.

2.

If B is a flat A-algebra and N is a flat B-module, then N is flat as an A-module.

Proof.

1.

If U→V→W is an exact sequence, then so is (U⊗M)→(V×M)→(W×M), hence (U⊗M)⊗N→(V⊗M)⊗N→(W⊗M)⊗N.

but (U⊗M)⊗N≅U⊗(M⊗N), qed.

2.

Let j:M→M′ be an injective A-module homomorphism. Since B is flat, (idB​⊗A​j):(B⊗A​M)→(B⊗A​M′) is injective. Consider (idB​⊗A​j) as a B-module homomorphism, then since N is flat, idN​⊗B​(idB​⊗A​j) is injective.

By associativity of tensor product, and N⊗B​B≅N, we have idN​⊗A​j injective, hence N is flat as A-module.

□

Exercise 9. Let 0→M′→M→M′′→0 be an exact sequence of A-modules. If M′ and M′′ are finitely generated, so is M.

Proof. Let f:M′→M and g:M→M′′ be the maps in the sequence. If x1​,…,xn​ generate M′ and y1​,…,ym​ generate M′′. For each yi​ select an element qi​∈M such that g(qi​)=yi​, then since M=⋃i=1m​(qi​+Kerg)=Imf+∑i=1m​(qi​), and Imf is generated by pi​=f(xi​), so M is generated by p1​,…,pn​,q1​,…,qm​.

□

Exercise 10. Let A be a ring, a an ideal contained in the Jacobson radical of A; let M be an A-module and N a finitely generated A-module, and let u:M→N be a homomorphism. If the induced homomorphism M/aM→N/aN is surjective, then u is surjective.

Proof. If uˉ:M/aM→N/aN is surjective, then forall y∈N there exists x∈M such that u(x)−y∈aN. That means, N=Imu+aN. So by Nakayama Lemma, N=Imu, i.e. u is surjective.

□

Exercise 11. Let A be a ring =0. Show that Am≅An⟹m=n.

Proof. let m be a maximal ideal of A, and ϕ:Am→An an isomorphism, then 1⊗ϕ:(A/m)⊗Am→(A/m)⊗An is an isomorphism between two A/m-vector spaces, hence the dim of two space are same, i.e. m=n.

□

Exercise 12. Let M be a finitely generated A-module and ϕ:M→An a surjective homomorphism. Show that Kerϕ is finitely generated.

Proof. Let x1​,…,xn​ be a set of generators of An, and y1​,…,yn​∈M such that ϕ(yi​)=xi​. Let M′ be the submodule generating by y1​,…,yn​, then clearly M′∩Kerϕ=0, and for all t∈M there exists y∈M′ such that f(t)=f(y), hence M′+Kerϕ=M. Summarize results above we get M≅M′⊕Kerϕ. Since M is finitely generated, Kerϕ must be finitely generated too.

□

Exercise 13. Let f:A→B be a ring homomorphism, and let N be a B-module. Regarding N as an A-module by restriction of scalars, form the B-module NB​=B⊗A​N. Show that the homomorphism g:N→NB​ which maps y to 1⊗y is injective and that g(N) is a direct summand of NB​.

Proof. Consider the quotient map B⊗A​N→B⊗B​N. Since B⊗B​N≅N, we have h:NB​→N which maps b⊗y to by.

Now h∘g=idN​, so g is injective. Consider map ϕ:NB​→N⊕Kerh defined by ϕ=h⊕(idNB​​−g∘h). (h(x−g(h(x)))=h(x)−h(x)=0 so the second part of image of ϕ is actually Kerh). We will prove ϕ is an isomorphism so NB​≅N⊕Kerh.

If ϕ(x)=0, i.e. h(x)=0 and x−g(h(x))=0, obviously x=0, so ϕ is injective. For any y∈N and x0​∈Kerh, let x=x0​+g(y), then h(x)=y and x−g(h(x))=x0​, hence ϕ is injective. All in all we have ϕ is an isomorphism so NB​≅N⊕Kerh.

□

Exercise 14. A partially ordered set I is said to be a direct set if for each pair i,j in I there exists k∈I such that i≤k and j≤k.

2.

μik​=μjk​∘μij​ whenever i≤j≤k.

Then the modules Mi​ and homomorphisms μij​ are said to form a direct system M=(Mi​,μij​) over the directed set I.

We shall construct an A-module M called the direct limit of the direct system M. Let C be the direct sum of Mi​, and identify each module Mi​ with its canonical image in C. Let D be the submodule of C generated by all elements of the form xi​−μij​(xi​) where i≤j and xi​∈Mi​. Let M=C/D, let μ:C→M be the projection and let μi​ be the restriction of μ to Mi​.

The module M, or MORE correctly the pair consisting of M and the family of homomorphisms μi​:Mi​→M is called the direct limit of the direct system M, and is written lim​Mi​. From the construction it is clear that μi​=μj​∘μij​ whenever i≤j.

Proof. No exercise here. For the last sentence, μi​(x)−μj​(μij​(x))=μ(x−μij​(x)), but x−μij​(x)∈D=Kerμ.

Exercise 15. In the situation of Exercise 14, show that every element of M can be written in the form μi​(xi​) for some i∈I and some xi​∈Mi​.

Show also that if μi​(xi​)=0 then there exists j≥i such that μij​(xi​)=0 in Mj​.

Proof. Any element of M can be written in form ∑i∈I​μi​(xi​) with only finite xi​ nonzero. But for any i,j there exists k such i,j≤k, so μi​(xi​)+μj​(xj​)=μk​(μik​(xi​)+μjk​(xj​)), hence any element of M can be written in form μk​(xk​) for some k.

If μi​(xi​)=0, i.e. xi​∈Kerμ=D, then we have

xi​=j,k∈I∑​(yj​−μjk​(yj​))=j∈I∑​zj​

where the sum contains only finite nonzero terms, and zj​ is projection to Mj​. But xi​∈Mi​ and the equation above is in a direct sum, so all elements zj​=0 except zi​=xi​. Select an index p∈I which ≥ any j,k appearing here, then

μip​(xi​)=j∈I∑​μjp​(zj​)=j,k∈I∑​(μjp​(yj​)−(μkp​∘μjk​)(yj​))=0

’Cause μjp​=μkp​∘μjk​ for any j≤k≤p.

Exercise 16. Show that the direct limit is charactered (up to isomorphism) by the following property. Let N be an A-module and for each i∈I let αi​:Mi​→N be an A-module homomorphism such that αi​=αj​∘μij​ whenever i≤j. Then there exists a unique homomorphism α:M→N such that αi​=α∘μi​ for all i∈I.

Proof. First prove (M,μi​) constructed here satisfies this condition.

For any N and αi​:Mi​→N satisfies αi​=αj​∘μij​, define β:C→N by β(∑i∈I​xi​)=∑i∈I​αi​(xi​), then for all i≤j and xi​∈Mi​ we have β(xi​−μij​(xj​))=0, hence D⊆Kerβ, so β induce an A-homomorphism α:M→N and αi​=α∘μi​ for any i∈I. Since all elements in M can be written in the form μi​(xi​), the map α is then unique.

If (M′,μi′​) is another system satisfying the condition, let N=M and αi​=μi​, there a unique homomorphism α:M′→M such that μi​=α∘μi′​. In the other direction there also exists a homomorphism β:M→M′ such that μi′​=β∘μi​, so μi​=α∘β∘μi​ for any i∈I. Again let N=M and αi​=μi​, there exists a unique homomorphism γ:M→M such that μi​=γ∘μi​ for any i. But both α∘β and idM​ meet the requirement of γ, so idM​=α∘β; and idM′​=β∘α vice versa. Hence α and β are inverse, and M≅M′.

□

Exercise 18. Let M=(Mi​,μij​),N=(Ni​,νij​) be direct systems of A-modules over the same directed set. Let M,N be the direct limits and μi​:Mi​→M,νi​:Ni​→N the associated homomorphisms.

Exercise 22. Let (Ai​,αij​) be a direct system of rings and let Ri​ be the nilradical of Ai​. Show that lim​Ri​ is the nilradical of lim​Ai​.

For the second part, if xy=0∈A, then there exists i∈I such that x=μi​(xi​) and y=μi​(yi​), hence μi​(xi​yi​)=0 and there exists j≥i that μij​(xi​yi​)=0. Since Aj​ is an integral domain, either μij​(xi​) or μij​(yi​) is zero, so in A either x or y is zero, and A is then an integral domain.

Flatness and Tor

Exercise 24. If M is an A-module, the following are equivalent:

1.

M is flat;

2.

TornA​(M,N)=0 for all n>0 and all A-modules N;

3.

Tor1A​(M,N)=0 for all A-modules N.

Proof. i) ⟹ ii): M is flat means ⊗M is exact functor. If P→N is a projective resolution of N, then P⊗M is exact, and TornA​(M,N)=Hi​(P⊗M)=0.

ii) ⟹ iii): Trivial.

iii) ⟹ i): Consider a exact sequence 0→N′→N→N′′→0, then there is a long exact sequence:

⋯→Tor1A​(M,N′)→→N′⊗M→N⊗M→N′′⊗M→0

If Tor1A​(M,N)=0 then ⊗M is exact, that is M is flat.

□

Exercise 25. Let 0→N′→N→N′′→0 be an exact sequence, with N′′ flat. Then N′ is flat ⟺ N is flat.

Proof. For all A-module M there exists a long exact sequence:

⋯→Tor2A​(M,N′′)→Tor1A​(M,N′)→Tor1A​(M,N)→Tor1A​(M,N′′)→⋯→0

If N′′ is flat, then Tor1A​(M,N′)≃Tor1A​(M,N). So N′ is flat ⟺Tor1A​(M,N′)=0 for all M⟺ Tor1A​(M,N)=0 for all M.

□

Now if M is generated by x1​,…,xn​, and let Mi​ be the submodule generated by x1​,…,xi​. Then there is exact sequence 0→Mi−1​→Mi​→Mi​/Mi−1​. So by the last exercise, it’s sufficient to show Tor1A​(Mi​/Mi−1​,N)=0. But Mi​/Mi−1​≃A/a for some ideal a, so it’s sufficient to show Tor1A​(A/a,N)=0 for all a.

By (2.19) again, we can only consider finitely generated ideals a.

Exercise 27. A ring A is absoutely flat if every A-module is flat. Prove that the following are equivalent:

1.

A is absoutely flat.

2.

Every principal ideal is idempotent.

3.

Every finitely generated ideal is a direct summand of A.

Proof. i) ⟹ ii): Let x∈A, then A/(x) is flat. Let f:(x)⊗A→A: f(x,a)=ax, tensor with A/(x) gives an injective map α:(x)⊗A/(x)→A/(x). By the commutative diagram, (x)⊗A→A→A/(x) (the zero map) equals (x)⊗A→(x)⊗A/(x)→A/(x). Since α is injective, Im((x)⊗A→(x)⊗A/(x))=0. So (x)⊗A/(x)=0 and (x)=(x2).

ii) ⟹ iii): For any x∈A, there exists a such that x=ax2, so e=ax is idempotent and (x)⊆(ex)⊆(e)⊆(x), i.e. (e)=(x). For any two idempotent e,f, we have (e,f)=(e+f−ef) cause e=e(e+f−ef). So any finitely generated ideal of A is idempotent principal ideal (e), and A=(e)⊕(1−e).

iii) ⟹ i): a⊕b=A, then A/a≃b. For all A-module N, Tor1A​(A/a,N)=Tor1A​(b,N). But b is a direct summand of A, hence flat. So Tor1A​(A/a,N)=0 for all finitely generated ideal a, and by Exercise 2.26 N is flat.

□

Exercise 28. A Boolean ring is absolutely flat. The ring of Exercise 1.7 is absolutely flat. Every homomorphic image of an absolutely flat ring is absolutely flat. If a local ring is absolutely flat, then it is a field.
