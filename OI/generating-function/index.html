生成函数

生成函数(generating function)又称母函数，是处理组合数学问题的一大利器。

普通型生成函数

Definition

对于一个数列 f=<f0,f1,f2⋯>f=<f_0,f_1,f_2\dots>f=<f0​,f1​,f2​⋯> ，构造形式幂级数

F(x)=∑i=0∞fixiF(x) = \sum_{i=0}^{\infty}f_ix^i F(x)=i=0∑∞​fi​xi

qi称为数列fff的普通型生成函数（ordinary generating function, 下简称OGF）。

于是

指数型生成函数

Definition

指数型生成函数（exponential generating function，简称EGF），其定义为

F(x)=∑nfnxnn!F(x)=\sum_nf_n\frac{x^n}{n!} F(x)=n∑​fn​n!xn​

其意义体现在其乘法操作上：

F(x)G(x)=∑n(∑i=0n(ni)fign−i)xiF(x)G(x)=\sum_n\left(\sum_{i=0}^n\binom nif_ig_{n-i}\right)x^i F(x)G(x)=n∑​(i=0∑n​(in​)fi​gn−i​)xi

多出来的(ni)\binom ni(in​)告诉我们它适用于排列的计算（相对于OGF的普通卷积对应组合）。

EGF的运算法则：

αF(x)+βG(x)=∑n(αfn+βgn)xnn!xmF(x)=∑n[n≥m]nm‾fn−mxnn!F(x)−∑i=0m−1fixixm=∑n1(n+m)m‾fn+mxnn!F(cx)=∑ncnfnxnn!F′(x)=∑nfn+1xnn!∫0xF(t)dt=∑n[n>0]fn−1xnn!F(x)G(x)=∑n(∑i=0n(ni)fign−i)xnn!\begin{aligned} \alpha F(x)+\beta G(x)&=\sum_n(\alpha f_n+\beta g_n)\frac{x^n}{n!}\\ x^mF(x)&=\sum_n[n\geq m]n^{\underline m}f_{n-m}\frac{x^n}{n!}\\ \frac{F(x)-\sum_{i=0}^{m-1}f_ix^i}{x^m}&=\sum_n\frac1{(n+m)^{\underline m}}f_{n+m}\frac{x^n}{n!}\\ F(cx)&=\sum_nc^nf_n\frac{x^n}{n!}\\ F^\prime(x)&=\sum_nf_{n+1}\frac{x^n}{n!}\\ \int_0^xF(t)\mathrm{d}t&=\sum_n[n>0]f_{n-1}\frac{x^n}{n!}\\ F(x)G(x)&=\sum_n\left(\sum_{i=0}^n\binom nif_ig_{n-i}\right)\frac{x^n}{n!} \end{aligned} αF(x)+βG(x)xmF(x)xmF(x)−∑i=0m−1​fi​xi​F(cx)F′(x)∫0x​F(t)dtF(x)G(x)​=n∑​(αfn​+βgn​)n!xn​=n∑​[n≥m]nm​fn−m​n!xn​=n∑​(n+m)m​1​fn+m​n!xn​=n∑​cnfn​n!xn​=n∑​fn+1​n!xn​=n∑​[n>0]fn−1​n!xn​=n∑​(i=0∑n​(in​)fi​gn−i​)n!xn​​
