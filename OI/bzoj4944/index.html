BZOJ4944 [NOI2017] 泳池

发表于 2018-03-19 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

有一个底边长 nnn 个单位，高为 +∞+\infty+∞ 的矩形网格，其中每个格子都有可能是危险的或者安全的。

每个格子安全的概率为 ppp ，危险的概率为 1−p1 - p1−p 。任意两个格子是否安全都是独立的。

要选取一个子矩形，要求它与底边相邻且其中所有格子都是安全的。最大化选取子矩形的面积。求答案恰好为 kkk 的概率。

n⩽109,k⩽1000n \leqslant 10^9, k \leqslant 1000n⩽109,k⩽1000 。

Solution

我们把“恰好为 kkk 的概率”转化为“小于 k+1k+1k+1 的概率减去小于 kkk 的概率”。

为了简化描述，下面把某一列从底边开始能够延伸出的最长的安全的格子长度称为其“安全高度”。

考虑计算“答案小于 kkk 的概率”。

考虑 DP ：令 fn,mf_{n, m}fn,m​ 表示底边长为 nnn ，且已知每一行安全高度都不小于 mmm 时的概率。
