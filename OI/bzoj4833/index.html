BZOJ4833 [Lydsy1704月赛] 最小公倍佩尔数

Description

令 (1+2)n=en+fn2(1+\sqrt2)^n=e_n+f_n\sqrt2(1+2​)n=en​+fn​2​，其中 en,fne_n, f_nen​,fn​ 都是整数。

再令 gn=lcmi=1nfig_n=\mathop{\rm lcm}_{i=1}^n f_ign​=lcmi=1n​fi​。

给定 n,pn, pn,p，求 (∑i=1nigi) mod p\left(\sum_{i=1}^n ig_i\right)\bmod p(∑i=1n​igi​)modp。

数据组数 T⩽210,n⩽106,2⩽p⩽109+7T\leqslant210,n\leqslant10^6,2\leqslant p\leqslant10^9+7T⩽210,n⩽106,2⩽p⩽109+7，ppp 是质数，f1…fnf_1\dots f_nf1​…fn​ 在  mod p\bmod pmodp 意义下不为 000。

Solution

根据容斥我们可以得到

lcmi∈Sfi=∏∅≠T⊆S(gcd⁡i∈Tfi)(−1)∣T∣−1\mathop{\rm lcm}_{i\in S} f_i=\prod_{\emptyset\neq T\subseteq S}\left(\gcd_{i\in T}f_i\right)^{(-1)^{|T|-1}} lcmi∈S​fi​=∅​=T⊆S∏​(i∈Tgcd​fi​)(−1)∣T∣−1

这实际上是 min-max 容斥

max⁡(S)=∑∅≠T⊆S(−1)∣T∣−1min⁡(T)\max(S)=\sum_{\emptyset\neq T\subseteq S}(-1)^{|T|-1}\min(T) max(S)=∅​=T⊆S∑​(−1)∣T∣−1min(T)

的应用。

另外，可以发现 gcd⁡(fi,fj)=fgcd⁡(i,j)\gcd(f_i,f_j)=f_{\gcd(i,j)}gcd(fi​,fj​)=fgcd(i,j)​ （实际上，所有 fi=Afi−1+Bfi−2,f0=0,f1=1f_i=Af_{i-1}+Bf_{i-2},f_0=0,f_1=1fi​=Afi−1​+Bfi−2​,f0​=0,f1​=1 且 gcd⁡(A,B)=1\gcd(A,B)=1gcd(A,B)=1 的序列 fff 都满足这个性质）。

于是

gn=lcmi=1nfi=∏∅≠T⊆Sfgcd⁡(T)(−1)∣T∣−1=∏i=1nfi∑i∣t≤nμ(t/i)=∏i=1n(∏d∣ifdμ(i/d))\begin{aligned} g_n=&\mathop{\rm lcm}_{i=1}^n f_i\\ =&\prod_{\emptyset\neq T\subseteq S}f_{\gcd(T)}^{(-1)^{|T|-1}}\\ =&\prod_{i=1}^nf_i^{\sum_{i\mid t\leq n}\mu(t/i)}\\ =&\prod_{i=1}^n\left(\prod_{d\mid i}f_d^{\mu(i/d)}\right) \end{aligned}gn​====​lcmi=1n​fi​∅​=T⊆S∏​fgcd(T)(−1)∣T∣−1​i=1∏n​fi∑i∣t≤n​μ(t/i)​i=1∏n​⎝⎛​d∣i∏​fdμ(i/d)​⎠⎞​​

所以求出所有的 fff 之后莫比乌斯反演出 ∏d∣ifiμ(i/d)\prod_{d\mid i}f_i^{\mu(i/d)}∏d∣i​fiμ(i/d)​，然后前缀积就可以得到 ggg 了。
