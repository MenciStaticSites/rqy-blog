LOJ3123 [CTS2019] 重复

发表于 2019-05-15 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

给定小写字母组成的字符串 sss 及正整数 mmm，求有多少小写字母组成的字符串 ttt 满足 ttt 的无穷重复中存在一个和 sss 长度相同但比 sss 字典序小的子串。

∣s∣,m≤2000|s|,m\leq2000∣s∣,m≤2000。

Solution

设 s=s1,ns=s_{1,n}s=s1,n​，即 n=∣s∣n=|s|n=∣s∣（我们用 sa,bs_{a,b}sa,b​ 表示从 aaa 到 bbb 的子串）。

首先我们观察到一个性质：

引理 1

如果 sss 的某个长为 ttt 的子串 si,i+t−1s_{i,i+t-1}si,i+t−1​ 字典序小于长为 iii 的前缀 s1,ts_{1,t}s1,t​，那么把 sss 替换成 s1,i+t−2s_{1,i+t-2}s1,i+t−2​ ，答案是不变的。

证明也不难：如果 ttt 的无穷重复中有某个子串 p<sp<sp<s，那么要么 p1,i+t−2<s1,i+t−2p_{1,i+t-2}<s_{1,i+t-2}p1,i+t−2​<s1,i+t−2​，要么 p1,i+t−2=s1,i+t−2p_{1,i+t-2}=s_{1,i+t-2}p1,i+t−2​=s1,i+t−2​ 而 pi+t−1,n<si+t−1,np_{i+t-1,n}<s_{i+t-1,n}pi+t−1,n​<si+t−1,n​。

对于前者，显然 p1,i−1p_{1,i-1}p1,i−1​ 也是 ttt 的无穷重复的子串；对于后者，可以发现必有 pi+t−1≤si+t−1p_{i+t-1}\leq s_{i+t-1}pi+t−1​≤si+t−1​，从而 pi,i+t−1≤si,i+t−1<s1,tp_{i,i+t-1}\leq s_{i,i+t-1}<s_{1,t}pi,i+t−1​≤si,i+t−1​<s1,t​，所以从 pip_ipi​ 这一位开头的字符串字典序一定比 si+t−2s_{i+t-2}si+t−2​ 小。□\square□

举个例子：如果 sss 为 egefd，由于 ef 大于 eg，因此如果有一个小于 sss 的串在前三位和 sss 相等的话第四位必定小于 s4s_4s4​ 也就是小于 f。那这样的话从这个串的第三位开头的串就是 e?*，其中 ?<d<g，因此这个串也一定比 ege 小。

同理我们也可以证明：如果 sss 有某个 border，那么可以把这个 border 删去。

我们这样处理过 sss 之后，它就满足：sss 的任意子串大于等于 sss 中与之等长的前缀；sss 的任意后缀严格大于与之等长的前缀。称满足第一个条件的串叫做可做串，两个都满足的叫强可做串（因为我只会做这样的串），那么我们要求出 sss 的最长可做前缀，并只考虑这个前缀。

至于如何处理，可以求一遍 KMP。假定 sss 的长为 i−1i-1i−1 的前缀是强可做的，要求它的长为 iii 的前缀是不是可做的。

假设有某个后缀 st,is_{t,i}st,i​ 小于等长的前缀 s1,i−t+1s_{1,i-t+1}s1,i−t+1​，那么肯定是 s1,i−t=st,i−1s_{1,i-t}=s_{t,i-1}s1,i−t​=st,i−1​ 而 si−1+1>sis_{i-1+1}>s_isi−1+1​>si​（因为前面的都是可做的）。也就是说是一个 border 拼上一个字符。

但是因为前面的串是可做的，可以证明 KMP 那种跳最长 border 过程中字符是非升的，所以只需要判 i−1i-1i−1 处的最长 border 即可。这样的话就可以得到最长的可做后缀；然后不停判断是否存在 border，如果存在就删去即可。

现在我们得到了一个可做串 sss。考虑用 26m26^m26m 减去不合法的情况，即从每个位置开始的子串字典序都大于等于 sss 的情况。大于等于 sss 等价于大于 pred(s)pred(s)pred(s)（最大的比 sss 字典序小的字符串）。

只考虑 n≤mn\leq mn≤m 的情况（如果 m>nm>nm>n，那么只需要看一个前 mmm 位恰好等于 s1,ms_{1,m}s1,m​ 的串的重复是否比 sss 小。根据 sss 的强可做性，这个是必然的。因此只需要算大于 sss 的情况即可，因为等于 sss 也是合法的）。

由于 sss 的后缀严格大于前缀，即 sn>s1s_n>s_1sn​>s1​，其 predpredpred 就是将末尾减一的结果（不需要退位）。可以发现得到的串必定是可做的，即任意子串大于等于等长的前缀。

以下我们设 sss 是可做的（就是之前那个 pred(s)pred(s)pred(s)），且 n=∣s∣<mn=|s|<mn=∣s∣<m，求有多少个长度为 mmm 的串 ttt 满足 ttt 的无穷重复的任意长度为 nnn 的子串都大于 sss。

ttt 的无穷重复中任意子串大于 sss，等价于说把 ttt 看成一个环之后，其任意长为 nnn 的子串字典序大于 sss。下面我们先把 ttt 看成一个环。称满足条件的环是合法环。

定义一个串 ppp 是好的，当且仅当 ∣p∣≤n|p|\leq n∣p∣≤n，且 p1,∣p∣−1=s1,∣p∣−1,p∣p∣>s∣p∣p_{1,|p|-1}=s_{1,|p|-1}, p_{|p|}>s_{|p|}p1,∣p∣−1​=s1,∣p∣−1​,p∣p∣​>s∣p∣​。

我们有如下引理：

引理 2

由若干个好串拼接而成的环 ttt 是合法环。并且对于任何一个合法环，将其表示成好串的拼接的方式是唯一的。

证明：引理有三个部分，我们分别证明：

  1. 先证明好串的拼接是合法的。如果有一个好串的拼接，对于其任意一个长为 nnn 的子串 qqq，qqq 一定有一个前缀 q1,kq_{1,k}q1,k​ 是某个好串的后缀。
    设这个好串为 ppp，那么 q1,k=p∣p∣−k+1,∣p∣>s∣p∣−k+1,∣p∣≥s1,kq_{1,k}=p_{|p|-k+1,|p|}>s_{|p|-k+1,|p|}\geq s_{1,k}q1,k​=p∣p∣−k+1,∣p∣​>s∣p∣−k+1,∣p∣​≥s1,k​。其中大于号是因为好串的前面部分和 sss 对应部分相等而最后一个字母大于 sss 的对应字母，
    大于等于号是因为 sss 的可做性。于是 q1,k>s1,kq_{1,k}>s_{1,k}q1,k​>s1,k​，显然有 q>sq>sq>s。
  2. 再证明合法环可以分解成好串的拼接。对于一个合法环，找到其最小表示法 ttt（即，其循环排列中字典序最小者），那么从开头开始对 sss 做匹配，每次发现某个位置比 sss 的相应位置大的时候划分出一个好串。
    只需要证明匹配到末尾的时候不可能还剩下没有匹配完的部分。如果到了末尾还没有匹配完，那么 ttt 必定有一个后缀与 sss 的某个前缀相等（就是匹配剩下的部分）。
    设 t∣t∣−k+1,∣t∣=s1,kt_{|t|-k+1,|t|}=s_{1,k}t∣t∣−k+1,∣t∣​=s1,k​，那么必定有 t1,k=s1,kt_{1,k}=s_{1,k}t1,k​=s1,k​，否则的话 ttt 的某个长度小于等于 kkk 的前缀是好串，而好串必定比 sss 字典序大，这和 ttt 是最小表示不符；
    但是既然 t1,k=s1,kt_{1,k}=s_{1,k}t1,k​=s1,k​，因为 ttt 实际上首尾是相邻的（是个环），所以 t∣t∣−k+1,∣t∣+t1,k=s1,k+s1,kt_{|t|-k+1,|t|}+t_{1,k}=s_{1,k}+s_{1,k}t∣t∣−k+1,∣t∣​+t1,k​=s1,k​+s1,k​ 也是其子串，因此必须要大于等于 s1,2ks_{1,2k}s1,2k​，即 s1,k≥sk+1,2ks_{1,k}\geq s_{k+1,2k}s1,k​≥sk+1,2k​，
    这样的话，根据 sss 的可做性又有 sk+1,2k≥s1,ks_{k+1,2k}\geq s_{1,k}sk+1,2k​≥s1,k​，于是 s1,k=sk+1,2ks_{1,k}=s_{k+1,2k}s1,k​=sk+1,2k​。这样的话必须有 t1,2k=s1,2kt_{1,2k}=s_{1,2k}t1,2k​=s1,2k​，因为 ttt 存在一个与 s1,2ks_{1,2k}s1,2k​ 相等的子串，而 ttt 是最小表示法从而 ≤s1,2k\leq s_{1,2k}≤s1,2k​，
    ttt 又是合法串因此 t1,2k≥s1,2kt_{1,2k}\geq s_{1,2k}t1,2k​≥s1,2k​。这样的话，重复之前的步骤，可以发现 s1,k=sk+1,2k=s2k+1,3k=…s_{1,k}=s_{k+1,2k}=s_{2k+1,3k}=\dotss1,k​=sk+1,2k​=s2k+1,3k​=…，而 t1,pk=s1,pkt_{1,pk}=s_{1,pk}t1,pk​=s1,pk​。
    当 ppp 充分大的时候，这说明 sss 必须无限长，这显然是矛盾的，因此原来的假设（ttt 最后匹配好串没有匹配完）是不成立的。
  3. 最后要证明好串拆分是唯一的。事实上如果固定起点，那么好串划分一定是唯一的，即上面的构造方式；而任选 ttt 的某个最小表示法，根据上面的证明容易发现不可能存在跨越首尾的好串，于是可以直接从这个点开始划分。

这样，我们就完成了引理的证明。□\square□
