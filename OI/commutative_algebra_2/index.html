Chapter 2. Modules 总结

模，模同态

固定一个环 AAA。一个 AAA-模 AAA-module 是一个阿贝尔群 MMM，且 AAA 在其上有一个线性的作用。换句话说，一个 AAA-模是一个二元组 (M,μ)(M,\mu)(M,μ)，其中 MMM 是一个阿贝尔群，而 μ\muμ 是一个 A×M→MA\times M\to MA×M→M 的映射，满足：

a(x+y)=ax+ay(a+b)x=ax+bx(ab)x=a(bx)1x=x\begin{aligned} a(x+y)&=ax+ay\\ (a+b)x&=ax+bx\\ (ab)x&=a(bx)\\ 1x&=x \end{aligned}a(x+y)(a+b)x(ab)x1x​=ax+ay=ax+bx=a(bx)=x​

其中 a,b∈A;x,y∈Ma,b\in A;x,y\in Ma,b∈A;x,y∈M。或者等价的，一个阿贝尔群 MMM 以及一个 A→E(M)A\to E(M)A→E(M) 的环同态（E(M)E(M)E(M) 是 MMM 的自同态环）。

AAA 本身，或者更一般的， AAA 上的任意理想 a\mathfrak{a}a 都可以看做 AAA-模。

一个 AAA-模之间的映射 f:M→Nf:M\to Nf:M→N 是 AAA-模同态 AAA-module homomorphism（或者说是 AAA-线性 AAA-linear 的）当且仅当 f(x+y)=f(x)+f(y);f(ax)=a⋅f(x)f(x+y)=f(x)+f(y);f(ax)=a\cdot f(x)f(x+y)=f(x)+f(y);f(ax)=a⋅f(x)。换句话说 fff 是一个群同态，并且它和 AAA 中元素的作用均交换。AAA-模同态的复合仍然是 AAA-模同态。

特别的，如果 A=kA=kA=k 是一个域，那么 kkk-模就等价于 kkk-线性空间，kkk-模同态就等价于 kkk-线性映射。

固定 AAA-模 N,MN,MN,M，则 N→MN\to MN→M 的所有同态也构成一个 AAA-模：只需定义 (f+g)(x)=f(x)+g(x);(af)(x)=a⋅f(x)(f+g)(x)=f(x)+g(x);(af)(x)=a\cdot f(x)(f+g)(x)=f(x)+g(x);(af)(x)=a⋅f(x) 即可。这个模记做 \Hom_A(N,M)；不引起歧义的时候有时候会省去下标 AAA。

同态 u:M′→Mu:M'\to Mu:M′→M 以及 v:N→N′v:N\to N'v:N→N′ 分别引导同态

\bar u:\Hom(M,N)\to\Hom(M',N);\qquad\bar v:\Hom(M,N)\to\Hom(M,N')

对任意的 MMM 存在一个自然的同构 \Hom(A,M)\cong M，因为任意同态 f:A→Mf:A\to Mf:A→M 唯一的由 f(1)f(1)f(1) 确定。

子模与商模

对任意 MMM 的子模 M′M'M′，存在一个从 M/M′M/M'M/M′ 的子模到 MMM 中包含 M′M'M′ 的子模的一一对应。（这句话哪里都有，群，环（理想），模…）
