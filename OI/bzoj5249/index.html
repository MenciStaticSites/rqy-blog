BZOJ5249 [多省联考2018] IIIDX

发表于 2018-04-08 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

给定一个长为 nnn 的序列 ddd （可能有重复）和一个常数 kkk （不一定是整数）。
求 ddd 的一个排列，要求满足 di⩾d⌊ik⌋d_i \geqslant d_{\lfloor\frac ik\rfloor}di​⩾d⌊ki​⌋​ 。

求所有满足条件的排列中字典序最大的一个。

Solution

首先，字典序最小的一个就是直接排序一遍（然而没什么用）。

贪心方法…算了我还是直接说正解好了。

假设我们有一个函数叫做 check ，能够在我填完前面一部分的时候告诉我是否存在合法解。

容易证明从前往后填到某一位的时候这一位填的越小越容易有解，所以可以逐位二分。

时间复杂度 O(nlog⁡n∗Check)O(n\log n * Check)O(nlogn∗Check) 。

先考虑一个 O(n)O(n)O(n) 的 check 怎么写。

显然 111 到 nnn 的顺序同时也是这棵树的 bfs 序，所以不存在一个点填了而它的父亲没填的情况。

考虑把所有填了的点（带着它的子树）从它的父亲上摘下来。

如果现在填了 kkk 个数，那么就有 kkk 棵树组成的森林。

可以发现，如果我给每棵树分配一些权值，使得每棵树里分配的权值都不小于这棵树的树根权值，并且个数恰好是这棵树里除根结点外的结点个数，
那么我一定能够找到一种合法的方案，即每棵树按 bfs 序从小到大填即可。

所以说我现在相当于有 kkk 个盒子，第 iii 个盒子有一个最小权值 tit_iti​ 和一个大小 sis_isi​ ，现在我要将剩余 n−kn-kn−k 个数放到这些盒子里，问是否存在合法方案。

那么根据贪心策略我只需要把盒子按 tit_iti​ 排序，然后把 n−kn-kn−k 个数从小到大放到盒子里即可。

这样就可以拿到 60 分（配合贪心有 70 ）的好成绩啦！

然后再来考虑如何把 check 加快。

我们令 bib_ibi​ 表示 i 这个数出现的次数，那么 check 返回 true 当且仅当

对于所有的 xxx ，都有 ∑i≥xbi≥∑ti≥xsi\sum_{i \ge x} b_i \ge \sum_{t_i \ge x} s_i∑i≥x​bi​≥∑ti​≥x​si​ 。

把不等式移项，可以得到它是一个后缀和大于零的形式。线段树维护最小后缀和，判断其是否非负即可。

总复杂度 O(nlog⁡2n)O(n\log^2n)O(nlog2n)
