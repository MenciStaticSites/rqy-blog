BZOJ3997 [TJOI2015] 组合数学

发表于 2018-03-14 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

一个n∗mn*mn∗m的网格图，每个格子要求了最少经过次数；每次可以向下/向右走，求至少走多少次可以使每个格子走过的次数不小于它的最小经过次数。n,m⩽1000n, m\leqslant1000n,m⩽1000

Solution

定理：有向图最小路径覆盖等于最大反链长。“反链”是指一些互不可达的点组成的集合。

那么这个问题中“互不可达”就相当于x1<x2,y1>y2x1<x2, y1>y2x1<x2,y1>y2这样，也就是左上到右下的一条路径。

dp即可。
