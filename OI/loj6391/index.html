LOJ6391 [THUPC2018] Tommy神的树

发表于 2018-06-04 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

一棵 nnn 个结点的树。初始时 aaa 和 bbb 是黑的，其他点是白的。

每次可以把某个黑点染成红的并把与它相邻的白点染成黑的。

问把结点染红的顺序有多少种。

1⩽a,b⩽n⩽2345671\leqslant a,b\leqslant n\leqslant2345671⩽a,b⩽n⩽234567

答案对 998244353998244353998244353 取模，时限 10s 。
