BZOJ3434 [WC2014] 时空穿梭

发表于 2018-03-14 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

有一个nnn维的超立方体[1,m1]×[1,m2]×⋯×[1,mn][1, m_1]\times[1, m_2]\times\dots\times[1,m_n][1,m1​]×[1,m2​]×⋯×[1,mn​]。

现在要再其内部选择ccc个整点，使得其每一维都是递增的（也即，∀i=1…c−1,j=1…n,xi+1,j>xi,j\forall i=1\dots c-1, j=1\dots n, x_{i+1, j}>x_{i,j}∀i=1…c−1,j=1…n,xi+1,j​>xi,j​），且这ccc个点共线。求方案数。

多组数据，至多100010001000组；n⩽11,c⩽20,mi⩽105n\leqslant 11, c\leqslant 20, m_i\leqslant 10^5n⩽11,c⩽20,mi​⩽105。

Solution

首先，我们发现只要满足xi<yi,i=1,2,…,nx_i < y_i, i = 1,2,\dots,nxi​<yi​,i=1,2,…,n ，（其中(x1,x2,…,xn),(y1,y2,…,yn)(x_1, x_2,\dots,x_n),(y_1, y_2,\dots,y_n)(x1​,x2​,…,xn​),(y1​,y2​,…,yn​)分别是第一个点和最后一个点的坐标）且点的不同排列算同一种即可。

那么，我们枚举这两个点之后，以其为端点的线段上共有（除去两端）gcd⁡(y1−x1,y2−x2,… .yn−xn)\gcd(y_1-x_1,y_2-x_2,\dots.y_n-x_n)gcd(y1​−x1​,y2​−x2​,….yn​−xn​)个点。其中选出c−2c-2c−2个点的方式有Cgcd⁡(y1−x1,y2−x2,… .yn−xn)c−2C_{\gcd(y_1-x_1,y_2-x_2,\dots.y_n-x_n)}^{c-2}Cgcd(y1​−x1​,y2​−x2​,….yn​−xn​)c−2​种，所以总方案数是

ans=\sum_{\substack{1\leqslant x_i < y_i\leqslant m_i \\ i = 1,2,\dots,n}}C_{\gcd(y_1-x_1,y_2-x_2,\dots,y_n-x_n)}^{c-2}

我们发现，所有pi=yi−xip_i=y_i-x_ipi​=yi​−xi​固定之后，方案数恰有

Cgcd⁡(p1,p2,…,pn)c−2∏i=1n(mi−pi)C_{\gcd(p_1, p_2,\dots,p_n)}^{c-2}\prod_{i=1}^n(m_i-p_i) Cgcd(p1​,p2​,…,pn​)c−2​i=1∏n​(mi​−pi​)

种，这是因为pip_ipi​固定后xix_ixi​只能取1,2,…,mi−pi1,2,\dots,m_i-p_i1,2,…,mi​−pi​.

于是我们有

ans=\sum_{\substack{1\leqslant q_i < m_i \\ i = 1,2,\dots,n}}C_{\gcd(p_1, p_2,\dots,p_n)}^{c-2}\prod_{i=1}^n(m_i-p_i)

优先枚举d=gcd⁡(p1,p2,…,pn)d=\gcd(p_1, p_2,\dots,p_n)d=gcd(p1​,p2​,…,pn​)（下式中mminm_{min}mmin​表示min⁡(m1,m2,…,mn)\min(m_1,m_2,\dots,m_n)min(m1​,m2​,…,mn​):

ans=\sum_{d=1}^{m_{min}-1}C_d^{c-2}\sum_{\substack{1\leqslant q_i < m_i \\ i = 1,2,\dots,n\\ \gcd(p_1, p_2,\dots,p_n)=d}}\prod_{i=1}^n(m_i-p_i)

令

f(d)=\sum_{\substack{1\leqslant q_i < m_i \\ i = 1,2,\dots,n\\ \gcd(p_1, p_2,\dots,p_n)=d}}\prod_{i=1}^n(m_i-p_i)

则有

\begin{aligned} \sum_{d\mid l}f(l)&=\sum_{\substack{1\leqslant q_i < m_i \\ i = 1,2,\dots,n\\d\mid\gcd(p_1, p_2,\dots,p_n)}}\prod_{i=1}^n(m_i-p_i)\\ &=\sum_{\substack{1\leqslant q_i'd < m_i \\ i = 1,2,\dots,n}}\prod_{i=1}^n(m_i-p_i'd)\\ &=\prod_{i=1}^n\sum_{p'=1}^{\left\lfloor\frac{m_i-1}d\right\rfloor}(m_i-p'd)\\ &=\prod_{i=1}^n\left[m_i\left\lfloor\frac{m_i-1}d\right\rfloor-\frac{d\left\lfloor\frac{m_i-1}d\right\rfloor\left(1+\left\lfloor\frac{m_i-1}d\right\rfloor\right)}2\right] \end{aligned}

最后一步是等差数列求和。

由莫比乌斯反演可知：

f(d)=∑d∣lμ(ld)∏i=1n[mi⌊mi−1l⌋−l⌊mi−1l⌋(1+⌊mi−1l⌋)2]f(d)=\sum_{d\mid l}\mu\left(\frac ld \right)\prod_{i=1}^n\left[m_i\left\lfloor\frac{m_i-1}l\right\rfloor-\frac{l\left\lfloor\frac{m_i-1}l\right\rfloor\left(1+\left\lfloor\frac{m_i-1}l\right\rfloor\right)}2\right] f(d)=d∣l∑​μ(dl​)i=1∏n​[mi​⌊lmi​−1​⌋−2l⌊lmi​−1​⌋(1+⌊lmi​−1​⌋)​]

于是有

ans=∑d=1mmin−1Cdc−2∑d∣lμ(ld)∏i=1n[mi⌊mi−1l⌋−l⌊mi−1l⌋(1+⌊mi−1l⌋)2]=∑l=1mmin−1∏i=1n[mi⌊mi−1l⌋−l⌊mi−1l⌋(1+⌊mi−1l⌋)2]∑d∣lμ(ld)Cdc−2\begin{aligned} ans&=\sum_{d=1}^{m_{min}-1}C_d^{c-2}\sum_{d\mid l}\mu\left(\frac ld \right)\prod_{i=1}^n\left[m_i\left\lfloor\frac{m_i-1}l\right\rfloor-\frac{l\left\lfloor\frac{m_i-1}l\right\rfloor\left(1+\left\lfloor\frac{m_i-1}l\right\rfloor\right)}2\right]\\ &=\sum_{l=1}^{m_{min}-1}\prod_{i=1}^n\left[m_i\left\lfloor\frac{m_i-1}l\right\rfloor-\frac{l\left\lfloor\frac{m_i-1}l\right\rfloor\left(1+\left\lfloor\frac{m_i-1}l\right\rfloor\right)}2\right]\sum_{d\mid l}\mu\left(\frac ld \right)C_d^{c-2} \end{aligned}ans​=d=1∑mmin​−1​Cdc−2​d∣l∑​μ(dl​)i=1∏n​[mi​⌊lmi​−1​⌋−2l⌊lmi​−1​⌋(1+⌊lmi​−1​⌋)​]=l=1∑mmin​−1​i=1∏n​[mi​⌊lmi​−1​⌋−2l⌊lmi​−1​⌋(1+⌊lmi​−1​⌋)​]d∣l∑​μ(dl​)Cdc−2​​

易知∏i=1n[mi⌊mi−1l⌋−l⌊mi−1l⌋(1+⌊mi−1l⌋)2]\prod_{i=1}^n\left[m_i\left\lfloor\frac{m_i-1}l\right\rfloor-\frac{l\left\lfloor\frac{m_i-1}l\right\rfloor\left(1+\left\lfloor\frac{m_i-1}l\right\rfloor\right)}2\right]∏i=1n​[mi​⌊lmi​−1​⌋−2l⌊lmi​−1​⌋(1+⌊lmi​−1​⌋)​]在将所有⌊mi−1l⌋\left\lfloor\frac{m_i-1}l\right\rfloor⌊lmi​−1​⌋作为常数看待的情况下是一个关于lll的不超过nnn次（是因为 mod 10007\bmod 10007mod10007之后有可能次数更低）的多项式，而且引起至少一个⌊mi−1l⌋\left\lfloor\frac{m_i-1}l\right\rfloor⌊lmi​−1​⌋改变的lll有O(nm)O(n\sqrt m)O(nm​)个，于是可以对每一个所有⌊mi−1l⌋\left\lfloor\frac{m_i-1}l\right\rfloor⌊lmi​−1​⌋都不改变的段分别求和。由于此时它是关于lll的多项式，我们只需预处理出所有的Sc,p,t=∑l=1tlp∑d∣lμ(ld)Cdc−2S_{c,p,t}=\sum_{l=1}^tl^p\sum_{d\mid l}\mu\left(\frac ld \right)C_d^{c-2}Sc,p,t​=∑l=1t​lp∑d∣l​μ(dl​)Cdc−2​即可。至于如何在只改变一个因式时快速维护乘积中各项的系数，可以用线段树实现（代码中为zkw线段树）。
