Description

给出一棵 nnn 个结点的树，要求给每个点一个正整数权值，使得相邻的结点的权值不相等，且最小化权值和。

n⩽104n\leqslant10^4n⩽104。

Solution

fi,j=∑kmin⁡t≠jfk,tf_{i,j}=\sum_k\min_{t\neq j}f_{k,t} fi,j​=k∑​t​=jmin​fk,t​

所以记录最小值、最小值对应的 jjj 和次小值即可。复杂度 O(n)O(n)O(n)。
