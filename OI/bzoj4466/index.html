BZOJ4466 [JSOI2013] 超立方体

发表于 2018-03-14 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

定义“超立方图”为：有2k2^k2k个点，以kkk位二进制数编号，两个点之间有边当且仅当它们的编号恰有一位不同。给出一个图，问它是否与“超立方图”同构。如果是，输出任意一种点与点的对应方案。(n⩽32768n \leqslant 32768n⩽32768)

Solution

先判断点数和边数、每个点的度数。

之后假设同构，求出同构方案后再check一遍。

显然超立方图的每个点都是对称的，于是我们任取一个点作为0，与它相邻的kkk个点编号为2i,i=0…k−12^i, i=0\dots k-12i,i=0…k−1。

之后，所有与000距离为222的点一定恰好与两个距离为111的点连边，且它的编号为这两个点编号的bit or。

与000距离为333的点的编号是所有与它相连的已知编号的点（距离一定是222）的bit or。

以此类推。bfs一遍即可。

最后不要忘记check。
