BZOJ3626 [LNOI2014] LCA

发表于 2018-03-14 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

nnn个点的树，qqq个询问，每个询问给出l,r,xl,r,xl,r,x，求∑i=lrdeplca(i,x)\sum_{i=l}^r dep_{lca(i, x)}∑i=lr​deplca(i,x)​。根的深度是111。n,q⩽50000n, q\leqslant 50000n,q⩽50000。

Solution

树剖？啥？能吃吗？

分块大法好！

将节点（按编号）每SSS个分一块，预处理出fi,j=∑x∈blockideplca(x,j)f_{i,j}=\sum_{x\in block_i}dep_{lca(x,j)}fi,j​=∑x∈blocki​​deplca(x,j)​，可以每个块dfs一遍。

然后对jjj这一维搞一个前缀和；再dfs一遍求dfs序（那种用来RMQ求LCA的），预处理ST表以O(1)O(1)O(1)LCA。

然后查询时整块直接前缀和相减；块内O(S)O(S)O(S)查询。

总时间复杂度O(n2S+qS)O(\frac{n^2}S+qS)O(Sn2​+qS)，SSS取nq\frac n{\sqrt q}q​n​时最小，为O(nq)O(n\sqrt q)O(nq​)。
