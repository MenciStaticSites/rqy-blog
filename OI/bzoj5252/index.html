BZOJ5252 [多省联考2018] 林克卡特树

发表于 2018-04-09 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

有一棵 nnn 个结点的树，每条边有边权。给定 0⩽k<n0\leqslant k\lt n0⩽k<n ，要求 k+1k+1k+1 条路径（可以仅包含单个点），使得它们不在结点处相交（包括两条路径端点相同的情况），最大化路径上的边权和。

0⩽k<n⩽3×1050\leqslant k \lt n \leqslant 3\times10^50⩽k<n⩽3×105

Solution

以下的所有 kkk 均指题目中的 k+1k+1k+1 （ 1⩽k⩽n1\leqslant k\leqslant n1⩽k⩽n　）。

首先，可以 DP 。令 fi,j,0/1/2f_{i,j,0/1/2}fi,j,0/1/2​ 表示在 iii 的子树中选择 jjj 条点不相交的路径的最大价值，其中 0/1/20/1/20/1/2 表示结点 iii 不在任意一条路径上 / 是某条路径的端点 / 在某条路径上（ 111 和 222 可能会重复，不过无所谓的）。

转移的时候考虑 iii 是否连向当前子树，跑几遍背包即可。

这样就能拿到 60 分的好成绩啦！

考虑如何优化。

~~严格的证明（我不会）~~打表找规律之后我们发现答案关于 kkk 是凸的。也就是说如果我令 gkg_kgk​ 表示答案的话 ggg 的差分序列是单调不增的。（ gk−gk−1≥gk+1−gkg_k-g_{k-1}\ge g_{k+1}-g_kgk​−gk−1​≥gk+1​−gk​ ）。

那么就可以上凸优化了。

凸优化是啥？

如果我们忽略掉 kkk 的限制而只去最大化路径边权和，显然可以 O(n)O(n)O(n) DP 。这时候，由于答案是凸的，所以它一定是先增后减的，我们求出的就是从增到减中间的那里的答案，也即差分序列从正到负那里的答案。

那么考虑如果我们能把差分序列整体加/减一个值，我们就能把这个“转折点”移动。如果我们把“转折点”恰好移动到 kkk 这个位置，那我们就能 O(n)O(n)O(n) DP 出 kkk 这个位置的答案。

如何把差分序列整体加/减一个值？很简单，只需要把 gig_igi​ 变成 gi−isg_i - isgi​−is ，就可以把差分序列整体减 sss 了。

那么既然差分序列是单调不增的，那 sss 越大，差分序列的“零点”也即原答案的“转折点”越靠左。既然我们要把“转折点”卡到第 kkk 个位置，就只需要二分 sss 即可。

每次二分之后 O(n)O(n)O(n) DP 出最大的 gi−isg_i - isgi​−is ，也就相当于不限制 k 但是每多选一条路径就会损失 sss 的权值（ sss 有可能是负的）；顺便还要求出对应的 iii ，即选了多少条路径（由于答案不是严格凸的，可能会有多种选法。这时候求出路径条数最少的即可）。

最后二分出 sss 再 DP 一遍，答案加上 ksksks 即可。

初始二分区间，由于差分不会超过直径长度也即不会超过正边权之和，所以上界为正边权之和即可。差分也不会小于最大正边权的相反数，所以下界为最大正边权的相反数即可。
（这两个性质可以通过求第一个差分和最后一个差分，因为差分是单调不增的）

复杂度 O(nlog⁡(nW))O(n\log(nW))O(nlog(nW)) ，其中 WWW 是边权最大值。
