CF947E Perpetual Subtraction

发表于 2018-04-28 更新于 2022-06-24 分类于 Solutions 阅读次数：

Description

有一个非负整数 xxx 。你要执行 mmm 次操作，每次操作是 x = randint(0, x) （ = 表示赋值， randint(0, x) 均匀随机地在 [0,x][0, x][0,x] 中取一个整数）。

现在已知初始的 xxx 会随机在 [0,n][0, n][0,n] 取值，且取 iii 的概率是 pip_ipi​ ，求最后取到 [0,n][0, n][0,n] 每个数的概率。

n⩽105,m⩽1018n\leqslant10^5, m\leqslant10^{18} n⩽105,m⩽1018

Solution

首先我们把概率都去掉：

如果我当前取到 i∈[0,n]i\in[0, n]i∈[0,n] 的概率是 fif_ifi​ ，变换之后概率是 fi∗f^*_ifi∗​ ，那么易知

fi∗=∑j=infjj+1f^*_i=\sum_{j=i}^n\frac{f_j}{j+1} fi∗​=j=i∑n​j+1fj​​

搞一个生成函数 F(x)=∑i=0nfixiF(x)=\sum_{i=0}^n f_ix^iF(x)=∑i=0n​fi​xi ，则

F∗(x)=∑i=0nxi∑j=0nfjj+1=∑j=0nfjj+1∑i=0jxi=∑j=0nfjj+1xj+1−1x−1=1x−1∑j=0nfj∫1xtjdt=∫1xF(t)dtx−1\begin{aligned} F^*(x)&=\sum_{i=0}^nx^i\sum_{j=0}^n\frac{f_j}{j+1}\\ &=\sum_{j=0}^n\frac{f_j}{j+1}\sum_{i=0}^j x^i\\ &=\sum_{j=0}^n\frac{f_j}{j+1}\frac{x^{j+1}-1}{x-1}\\ &=\frac1{x-1}\sum_{j=0}^n f_j\int_1^x t^j{\rm d}t\\ &=\frac{\int_1^x F(t){\rm d}t}{x-1} \end{aligned}F∗(x)​=i=0∑n​xij=0∑n​j+1fj​​=j=0∑n​j+1fj​​i=0∑j​xi=j=0∑n​j+1fj​​x−1xj+1−1​=x−11​j=0∑n​fj​∫1x​tjdt=x−1∫1x​F(t)dt​​

这个东西很不好：首先它积分下限是 111 而不是 000 ；其次它除以 x−1x-1x−1 而不是 xxx 。

那么我们可以想到令 G(x)=F(x+1)G(x)=F(x+1)G(x)=F(x+1) ，则

G∗(x)=F∗(x+1)=∫1x+1F(t)dtx+1−1=∫0xG(t)dtxG^*(x)=F^*(x+1)=\frac{\int_1^{x+1}F(t){\rm d}t}{x+1-1}=\frac{\int_0^x G(t){\rm d}t}x G∗(x)=F∗(x+1)=x+1−1∫1x+1​F(t)dt​=x∫0x​G(t)dt​

这个就很好：换成数列形式就是 gi∗=gii+1g^*_i=\frac{g_i}{i+1}gi∗​=i+1gi​​ 。

所以进行 mmm 次操作就是 gi∗=gi(i+1)mg^*_i=\frac{g_i}{(i+1)^m}gi∗​=(i+1)mgi​​ 。

于是我们只需要考虑如何由 fff 变换成 ggg 和如何逆变换即可。

由于

∑i=0ngixi=∑j=0nfj(x+1)j=∑j=0nfj∑i=0j(ji)xi\sum_{i=0}^ng_ix^i=\sum_{j=0}^nf_j(x+1)^j=\sum_{j=0}^nf_j\sum_{i=0}^j{j\choose i}x^i i=0∑n​gi​xi=j=0∑n​fj​(x+1)j=j=0∑n​fj​i=0∑j​(ij​)xi

所以

gi=∑j=in(ji)fjg_i=\sum_{j=i}^n{j\choose i}f_j gi​=j=i∑n​(ij​)fj​

即

i!gi=∑j=inj!fj(j−i)!i!g_i=\sum_{j=i}^n\frac{j!f_j}{(j-i)!} i!gi​=j=i∑n​(j−i)!j!fj​​

i!fi=∑j=1nj!fj(−1)j−i(j−i)!i!f_i=\sum_{j=1}^n\frac{j!f_j}{(-1)^{j-i}(j-i)!} i!fi​=j=1∑n​(−1)j−i(j−i)!j!fj​​
